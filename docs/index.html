每日執行流程

  1. 賽前取得當日賽程、先發投手、近期戰績與盤口。
  2. 第一層只做勝方預測，先追蹤模型準確率與信心值。
  3. 若確認模型與主模型方向相反，該場標記為一般預測，不進入高信心名單。
  4. 等真實盤口與成交賠率補齊後，再啟用投注金額、PnL 與 ROI 計算。
  5. 每日結束先更新預測正確率；盤口層完成後再更新投注績效。

實戰注意事項

  • 此專案是投注紀律與模型驗證工具，不是獲利保證。
  • 目前優先統計真實勝方預測準確率；投注 ROI 等盤口資料補齊後再啟用。
  • 若資料來源、盤口或先發投手變動，應先更新資料再產出每日預測。

# MLB 投注計畫上網方式

這個專案已整理成 GitHub Pages 靜態網站部署格式。

## 會上網的內容

只會發布 `docs/` 裡的網頁：

- `index.html`
- `daily_predictions.html`
- `betting_ticket.html`
- `totals_predictions.html`
- `advanced_factors.html`
- `backtest_2026_report.html`
- 其他報表頁

## GitHub Pages 設定

上傳到 GitHub 後，到 repository：

1. `Settings`
2. `Pages`
3. `Build and deployment`
4. `Source` 選 `GitHub Actions`

之後每次推送到 `main` 或 `master`，`.github/workflows/deploy-pages.yml` 會自動把 `docs/` 發布到網路。

## 本機更新後發布

先更新今日資料：

```powershell
.\立刻更新一次.cmd
```

確認網頁：

```powershell
.\開啟網頁.cmd
```

再提交並推送：

```powershell
git add .
git commit -m "Update MLB betting site"
git push
```

如果尚未有 GitHub 遠端，需先建立 GitHub repository，然後設定 remote：

```powershell
git remote add origin https://github.com/你的帳號/你的倉庫.git
git branch -M main
git push -u origin main
```
